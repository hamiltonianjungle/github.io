Home

Welcome to the Hamiltonian Jungle — an online resource for categorising the complexity classifications of various Hamiltonian problems. Please note, this is a work in progess! If you have any suggestions please refer to the Contact page for more information.

The current bulk of the work is concerned with the local Hamiltonian problem. We are of course open to including any an all Hamiltonian related problems. We also note that given the current standing of the Jungle, if not specified, the interaction strengths found in Hamiltonians can be assumed to follow $J_{ij} \leq \mathsf{poly}(n)$.
